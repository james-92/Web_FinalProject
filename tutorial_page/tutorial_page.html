Chest

胸推機

1.挑選一個不會作完第一組就過度疲勞的重量
2.肩胛骨保持中立位的同時虎口發力感受上胸刺激
3.上提至胸前即可停滯3秒再慢慢放回起始位置


平板啞鈴推舉

1.將啞鈴直立放置於膝蓋上
2.用腳輔助將啞鈴抬起順勢向後躺
3.調整重心，肩胛收緊讓背部確實貼合平板，腹部收縮
4.手肘與身體保持同一個平面，前臂與身體保持90度
5.將啞鈴上推至胸部上方慢慢放回起始位置並停滯3秒

普羅米修斯世紀組

1.挑選一個相對輕鬆的重量
2.坐在椅子前緣，身體向後傾斜
3.虎口發力向前推停滯一秒回到起始位置
4.一組坐50次中間休息10~15秒，共兩組

啞鈴下斜臥推

1.躺在平板上，視線與啞鈴水平線垂直
2.虎口握在啞鈴稍與肩同寬處
3.調整重心，肩胛收緊讓背部確實貼合平板，腹部收縮
4.將槓鈴推起，保持啞鈴運動方向與下胸垂直
5.下放至下胸位置停滯一秒返回起始位置

上斜啞鈴臥推

1.將啞鈴直立放置於膝蓋上
2.用腳輔助將啞鈴抬起順勢向後躺
3.調整重心，肩胛收緊讓背部確實貼合平板，腹部收縮
4.可適當將身體微微拱起避免肩部肌肉參與
5.手肘與身體保持同一個平面，前臂與身體保持90度
6.將啞鈴上推至上胸位置慢慢放回起始位置並停滯3秒

Arm

二頭彎舉

1.雙腳與肩同寬，後背、頭貼在牆上
2.雙手彎舉至胸前，手臂微微外旋

單邊集中彎舉

1.坐在凳子上將手臂緊貼大腿內側
2.大臂不動的情況下向上彎舉，同時外旋手臂
3.頂峰保持0.5秒後還原

繩索下壓

1.手肘固定，感受三頭收縮
2.垂直下壓，不要往身體內側拉
3.下壓至底部停留3秒收縮

啞鈴側平舉

1.向前微微俯身
2.手握啞鈴感覺手肘向上帶起整個手臂
3.頂峰停滯3秒慢慢放回

FacePull

1.向前微微俯身
2.手握啞鈴感覺手肘向上帶起整個手臂
3.頂峰停滯3秒慢慢放回

Back

椅上正手划船

1.選擇配重:選擇合適重量的啞鈴
2.調整椅子:將bench調整至傾斜約45度的狀態
3.進入準備姿勢:維持脊椎排列與下背打直， 將上半身貼平於bench上並且頭部不可靠在椅上(頸部須維持脊椎排列)
4.划船: 由背部肌肉引導將啞鈴提起，至少收縮至手肘與身體平行
5.以一組8下為目標
6.完成後可將啞鈴直接放手使其落地或是移動至其他地方並平放

cable下拉

1.選擇配重:將插銷插入合適的配重
2.調整握距:握距應比肩寬寬1~2個拳頭左右
3.將橫桿下拉:肩部放鬆並下拉，並坐直後稍微將胸口挺出
4.開始下拉:先收縮肩胛骨，在收縮背部使橫桿停於胸口處，回程先放鬆背部再放鬆肩胛
5.以一組8下為目標

cable後拉

1.選擇配重:將插銷插入合適的配重
2.調整握距:握距應比肩寬寬1~2個拳頭左右
3.將橫桿下拉:肩部放鬆並下拉，並坐直後稍微將胸口挺出
4.開始後拉:先收縮肩胛骨，在收縮背部使手腕貼齊於，回程先放鬆背部再放鬆肩胛
5.以一組8下為目標

引體向上

1.將手握至比肩寬寬1~2個拳頭左右
2.可稍微向上傾斜將胸口挺出，腳步可打直或是盤起
3.先收縮肩胛，再收縮背部
4.以一組8下或是力竭為目標
5.可使用彈力帶輔助(將彈力帶綁於兩邊後先踩上一邊並握好握把後在將另一腳放上)

地雷管

1.選擇配重:將插銷插入合適的配重
2.調整握距:握距應與肩膀同寬
3.開始划船:將雷管拉起並收縮至底部時停頓一下再回去
4.以一組8下為目標


Leg

自由槓深蹲

1. 起槓: 站距比肩膀窄，將脊椎排列好後將肩胛骨固定後吸氣到丹田處並核心發力。
2. 順著重心移出掛勾:先將整體重心向後移動順著重心把腳步往後挪。
3. 進入準備姿勢:站距站開到至少與肩膀同寬(腳內側不可超過肩膀)可稍微外八，
將重心壓到腳掌並直視前方，膝蓋稍微外旋並收肩胛壓髂，吸氣到丹田處並核心發力。
4. 開始深蹲:過程中不可吐氣必須將氣憋好，直視前方並且確保胸口與膝蓋擴張不內塌，並在觸底時馬上起來站直，回到準備姿勢。
5. 回槓:直線向前走並確定走到底部後將髂向後退。

啞鈴高腳杯深蹲

1. 拿起配重: 站距站開到至少與肩膀同寬(腳內側不可超過肩膀)可稍微外八，將脊椎排列好後將手掌托於配重下將其托起，手肘需緊貼身體配重需緊貼胸口。
2. 順著重心移出掛勾:先將整體重心向後移動順著重心把腳步往後挪。
3. 進入準備姿勢:站距站開到至少與肩膀同寬(腳內側不可超過肩膀)可稍微外八，
將重心壓到腳掌並直視前方，膝蓋稍微外旋並收肩胛壓髂，吸氣到丹田處並核心發力。
4. 開始深蹲:過程中不可吐氣必須將氣憋好，直視前方並且確保胸口與膝蓋擴張不內塌，並在觸底時馬上起來站直，回到準備姿勢。
5. 結束:直線向前走並將配重放回

腿推機

1. 選擇配重:將合適的配重配置到腿推機上
2. 注意事項:切記不可將腳完全伸直(避免膝蓋反折)，且每下需達最低深度
3. 站位選擇:不同站位刺激部位不同，選擇好站位後啟動機器
4. 使用機器:將腳推至接近伸直後讓重量下壓，並達最低深度後推回
5. 機器固定:做完後將機器卡榫卡回

壺鈴單腿蹲

1. 選擇配重:將合適的配重的壺鈴
2. 站位選擇:站距與平時走路相同
3. 開始單腿蹲:向前邁出一步使後腳膝蓋成90度
4. 回到準備姿勢:向後或是向前站起回到直立狀態


硬舉

1. 選擇配重:選擇合適配重配置到橫槓上
2. 腳步站距調整:將腳開至與與肩同寬。
3. 進入準備姿勢:站距站開到至少與肩膀同寬(腳內側不可超過肩膀)可稍微外八，將重心壓到腳掌並直視前方，膝蓋稍微外旋並收肩胛壓髂，吸氣到丹田處並核心發力。
4. 開始硬舉:過程中不可吐氣必須將氣憋好，直視前方並且確保胸口與膝蓋擴張不內塌，由臀部發力並將槓鈴舉起，在頂端時進入Lock姿勢。
5. 回槓:由臀部先向後退並將槓鈴直線落下。
